\# Home Network Security \& Management Project



Conducted a security assessment of a home network, documenting architecture and recommending improvements in patch management, access control, and network security.





\## Project Summary



Analyzed and secured a small-scale network using enterprise cybersecurity principles.



This project demonstrates hands-on experience with:



\* Network security architecture

\* Vulnerability scanning and analysis

\* Endpoint hardening

\* Patch management

\* Access control implementation



Built to simulate real-world IT and SOC responsibilities.



\---



\## Overview



This project transforms an undocumented and minimally secured network into a structured, manageable, and defensible environment using industry best practices.



\---



\## Objectives



\* Document all network devices, protocols, and architecture

\* Improve network security posture

\* Implement monitoring and vulnerability detection tools

\* Apply cybersecurity best practices (least privilege, patching, backups)

\* Create a scalable and maintainable network plan



\---



\## Skills Demonstrated



\* Network Security Architecture

\* Vulnerability Scanning (Nmap)

\* Packet Analysis (Wireshark)

\* IDS/IPS Concepts (Snort)

\* Patch Management

\* Access Control \& Least Privilege

\* Network Documentation \& Change Management



\---



\## Results \& Outcomes



\* Reduced attack surface by limiting open ports and tightening firewall rules

\* Implemented structured patch management across all devices

\* Established secure backup strategy (cloud + offline)

\* Improved visibility into network activity using scanning tools

\* Created full network documentation for ongoing management

\## Security Enhancements Implemented



\### Endpoint Security



\* Configured Windows Defender Firewall with stricter rules

\* Deployed antivirus protection across endpoints

\* Enabled automatic system updates



\### Access Control



\* Restricted admin account to maintenance use only

\* Enforced standard user privileges

\* Planned multi-factor authentication (MFA)



\### Backup Strategy



\* Implemented cloud backup (OneDrive)

\* Configured local backup (external SSD)

\* Established weekly offline backups



\### Physical Security



\* Secured network hardware in locked location

\* Planned surveillance system implementation



\---



\## Patch Management Strategy



\* Enabled automatic updates for all endpoints

\* Applied updates to:



&#x20; \* Windows systems

&#x20; \* Mobile devices

&#x20; \* Console firmware

\* Conducted monthly verification of patch status

\* Performed quarterly review for end-of-life systems



\---



\## User \& Access Management



\* Enforced least privilege model

\* Restricted administrative access

\* Configured account lockout after failed login attempts

\* Planned centralized identity management (Azure AD)



\---



\## Risk \& Limitations



\* No centralized server for automation

\* Limited network segmentation

\* Some enterprise features simulated in a home environment
